Set Shapes 101

设置形状101

These are shapes that usually already have names in everyday use. They are a box (or cuboid), a sphere, a cylinder, a cone, regular polygons, a plane and a specialist horizontal plane called the ground. Slightly less well known but also included in set shapes are a torus, a torus knot and the polyhedra. You will have to wait for the next section to learn how to create shapes that have no everyday names but are formed through data sets and parameters. These are termed parametric shapes.

这里有一些已经在日常生活中命名的形状。它们是是盒子（或者说长方体）、球体、圆柱体、椎体、正多边形、平面以及一种特别的叫做地面的水平平面。没有那么知名但同样包括的形状有环面、环面节以及多面体。你可能必须等到后面的教程才能学到如何建立那些没有日常命名的形状，但是它们也是通过一系列的参数生成的。所有这些形状都称为“参数化形状”。

In the 101 course you will only meet a limited number of set shapes, starting on this page with boxes, spheres, planes and ground. Also you will just use the MeshBuilder method of shape creation rather than the older legacy Mesh method. How to create all the set shapes using either method and the advantages and disadvantages of both can be found by doing Further Reading.

在这个101教程中，你只会遇到有限的几种预设形状，开始首先是盒子、球体、平面和地面。另外你只需使用建立形状的MeshBuilder方法代替旧的遗留Mesh方法。如何使用相应的方法建立所有的预设形状，以及对应的优点和缺点都可以在“扩展阅读”中找到

MeshBuilder Method

MeshBuilder方法

The general form to create a set shape is:

建立一个预设形状的通用方法是：

var shape = BABYLON.MeshBuilder.CreateShape(name, options, scene);

The options parameter allows you to do such things as set the size of the shape and set whether you can update it or not. Specific examples below.

这里的options参数允许你设置形状的尺寸以及是否可以更新这个形状。针对性的例子如下.

  • Playground Example of a Plane - 一个平面的Playground例子

 

sourcePlane is a unique option for a plane mesh and provides a method to orientate and position it. For now just consider its orientation which on creation is the vector (0, 0, 1). Now should you want the orientation to be the vector (0, -1, 1) then you create a source plane using

源平面是一个平面网格独有的选项，它提供了一个为平面网格定向和定位的方法。你现在只需认为平面网格刚建立时的朝向为向量(0, 0, 1)，现在如果你希望这个朝向变为(0, -1, 1)，那么你可以使用以下方法建立一个源平面：

var sourcePlane = new BABYLON.Plane(0, -1, 1, 0);
sourcePlane.normalize();

This creates a mathematical plane which is used as the orientation source. The fourth parameter is a distance moved in the direction of the orientation vector. At this stage leave as 0.

这个方法建立了一个数学意义上的平面，这个平面被作为确定平面网格方向的数据源（以这个源平面为参数建立平面网格，则建立的平面网格将继承源平面的朝向和据原点距离）。第四个参数表示数学平面从原点出发沿着平面朝向移动的距离。在此阶段，保留为0。

  • Playground Example with Source Plane - 使用源平面的Playground例子

var createScene = function () {

 

    // Create the scene space

    var scene = new BABYLON.Scene(engine);

 

    // Add a camera to the scene and attach it to the canvas

    var camera = new BABYLON.ArcRotateCamera("Camera", -Math.PI / 2, Math.PI / 2, 4, BABYLON.Vector3.Zero(), scene);

    camera.attachControl(canvas, true);

 

    // Add lights to the scene

    var light1 = new BABYLON.HemisphericLight("light1", new BABYLON.Vector3(1, 1, 0), scene);

    var light2 = new BABYLON.PointLight("light2", new BABYLON.Vector3(0, 1, -1), scene);

 

    var sourcePlane = new BABYLON.Plane(0, -1, 1, 0);

    sourcePlane.normalize();

   

    // Add and manipulate meshes in the scene

    var plane = BABYLON.MeshBuilder.CreatePlane("plane", {height:2, width: 1, sourcePlane: sourcePlane, sideOrientation: BABYLON.Mesh.DOUBLESIDE}, scene);

 

    return scene;

 

};

A variation on CreateGround is CreateGroundFromHeightMap which lets you form undulating ground rather than a flat plane.

CreateGround的一个变异是“CreateGroundFromHeightMap”，这个方法允许你使用一张图片建立高低起伏的地面而不是一个平面。

Face Colors or UV

面颜色或者纹理坐标

This is only available on a limited number of meshes that have distinct faces such as a box but not a sphere. This allows you to give each face for those meshes an individual color or image. Find out about Face Colors and UV.

这个特性只对有限的几种具有不同面的网格有用，比如盒子可以但球体不行。这个特性允许你给这些网格的不同面赋予不同的颜色或者图片。在这里查看关于面颜色和纹理坐标的内容。

Updatable

可更新

Where a mesh has the updatable parameter set to true it means that it is possible to alter the data associated with each vertex of the mesh and so alter the shape of the mesh. For more information see How to Update Vertices

一个网格的可更新参数被设为true，意味着可以通过改变和网格中每个顶点相关的数据来改变网格的形状。更多的信息请看“如何更新顶点”

Side Orientation

面朝向

The side orientation option is used to change how each side of a mesh is viewed.

面朝向选项用来设置网格的各个面是否可见

There are four possible values for this option :

这个选项有四种可能的值

  • BABYLON.Mesh.FRONTSIDE, 只显示正面
  • BABYLON.Mesh.BACKSIDE, 只显示背面
  • BABYLON.Mesh.DOUBLESIDE, 两面都显示
  • BABYLON.Mesh.DEFAULT which is the default value and equals FRONTSIDE currently. 这个是默认值，并且目前等于正面

 

In the following examples you can compare between DEFAULT and DOUBLESIDE by moving your screen pointer left and right to rotate the planes.

Front and Back UV

正面和背面纹理坐标

When a mesh has a sideOrientation option present and it is set to DOUBLESIDE then it is possible for the front and back to display different images. For more information see How To Display Front and Back Images

当一个网格的面朝向选项存在，并被设为双面时，其正面和背面可能显示不同的图片。更多的信息请看“如何显示正面和背面图片”。
